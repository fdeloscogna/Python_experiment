PyTorch project for human centred data classification that combines deep learning experimentation with practical machine learning engineering. 
It includes custom dataset creation, image preprocessing and augmentation, transfer learning with pretrained CNNs, GPU-aware training, and evaluation through accuracy, F1 score, recall, confusion matrices, and runtime analysis. 
The aim is to compares the pre trained model like ResNet50, VGG16, and GoogLeNet on a three-class task and also explores validation and class-balancing strategies such as Stratified K-Fold and weighted sampling.
